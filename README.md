#  AI Assistant GUI using Python (Tkinter)

This project is a **Python-based GUI application** that allows users to search content on **YouTube**, **Google**, and **Instagram** directly from a single interface.

---

## Features
- Simple and clean GUI using Tkinter
- Search videos on YouTube
- Search queries on Google
- Open Instagram user profiles
- Automatically opens results in the web browser

---

##  Technologies Used
- Python
- Tkinter (GUI)
- webbrowser module

---

##  Installation

No external libraries required.  
Tkinter comes pre-installed with Python.
